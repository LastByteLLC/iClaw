# Changelog

All notable changes to iClaw are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

When releasing, add a section for the new version below `## [Unreleased]`.
The release workflow extracts the section under `## [X.Y.Z]` and uses it for
both the GitHub Release notes and the Sparkle appcast description.

## [Unreleased]

## [1.0.0] - 2026-04-27

- Initial release
- On-device AI agent powered by Apple Foundation Models
- 30+ built-in tools: weather, stocks, calendar, email, web search, and more
- Liquid Glass menu bar HUD
- Voice input with on-device speech recognition
- Safari browser extension
- Dynamic widget system
- Vector memory with GRDB + sqlite-vec
