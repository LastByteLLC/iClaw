iClaw Help

Getting Started

Opening iClaw

iClaw lives in your menu bar. Click the claw icon to toggle the floating HUD. The HUD stays on top of all windows and does not steal focus from your current app.

Asking a question

Type your request in the text field and press Return. iClaw automatically routes your query to the right tool. For example:

  • "What's the weather in Tokyo?" — opens Weather
  • "Convert 50 USD to EUR" — opens Convert
  • "Set a timer for 10 minutes" — opens Timer
  • "$AAPL" — opens Stocks for Apple

Follow-up questions

After a tool responds, a context pill appears above the input field showing the prior tool and entity (e.g. "Weather · Tokyo"). Short follow-ups like "What about tomorrow?" are automatically routed to the same tool.

Tap the context pill to anchor it — this locks the tool for your next message. The pill auto-dismisses after 30 seconds or when you type a longer message.

Paste suggestions

When you copy content to the clipboard, iClaw may show suggestion pills above the input field based on what you copied. Tap a suggestion to run that action on the clipboard content.

File attachments

Drag and drop an image or audio file onto the HUD. iClaw detects the file type and suggests relevant tools, such as transcription for audio or description for images.

iClaw requires Apple Intelligence. Make sure it is enabled in System Settings > Apple Intelligence & Siri. You can also use Ollama as an alternative backend — see Settings > Models.
