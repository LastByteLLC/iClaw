iClaw Help

Conversational Modes

iClaw has special conversational modes for tasks that require back-and-forth interaction. When a mode is active, a colored indicator appears and the conversation stays focused on that task.

Research Mode

Triggered by phrases like "research this topic" or "do a deep dive on...". iClaw performs multi-source web research and presents findings with citations. Ask follow-up questions to refine the research.

Rewrite Mode

For iterative text editing. Paste text and ask iClaw to rewrite, shorten, formalize, or improve it. Each follow-up refines the previous version.

Automate Mode

A step-by-step guide for creating AppleScript automations. Describe what you want to automate and iClaw walks you through building it. Available in the DMG version only.

Tech Support Mode

Interactive diagnostics for Mac issues. Describe your problem and iClaw asks targeted questions, checks system status, and suggests fixes.

Rubber Duck Mode

A thinking-out-loud companion for debugging. Explain your problem step by step — iClaw asks clarifying questions to help you find the answer yourself.

Exiting a mode

Each mode has exit phrases (like "done", "exit", or "thanks"). You can also start a completely different request to leave the current mode.
