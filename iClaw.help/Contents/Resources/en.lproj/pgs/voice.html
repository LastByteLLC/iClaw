iClaw Help

Voice Input & Speech

Voice input

Click the microphone button in the input field to speak your request. Speech recognition runs entirely on-device using Apple's built-in transcription engine.

The microphone button is hidden if speech recognition is unavailable on your system.

Text-to-speech

iClaw can read responses aloud. Each message bubble has a speaker button to read that individual response.

To have iClaw automatically read every response, enable Auto-Speak in Settings > General > Speech.

Voice selection

You can choose which voice iClaw uses for text-to-speech in Settings. Options include the system default voice and any Premium or Enhanced voices you have installed.

To install additional voices, go to System Settings > Accessibility > Spoken Content > System Voice > Manage Voices.
