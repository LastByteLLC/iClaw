iClaw Help

Browser Extension

The optional browser extension lets iClaw fetch web pages using your browser's real cookies and sessions. This means iClaw can access pages that require login, such as paywalled articles or authenticated dashboards.

How it works

When the extension is connected:

  1. iClaw sends a URL to the extension via a local Unix socket
  2. The extension fetches the page in your browser (with your cookies)
  3. The page content is sent back to iClaw for processing

Without the extension, iClaw falls back to a built-in web view or plain HTTP requests.

Supported browsers

  • Google Chrome
  • Mozilla Firefox
  • Safari (via App Extension)

Setup

Enable the browser bridge in Settings > General > Browser Bridge. The connection status indicator shows whether the extension is connected.

All communication between iClaw and the browser extension happens locally on your Mac via a Unix socket. No data is sent to any server.
