iClaw Help

Settings

Open Settings from the iClaw menu bar icon or by asking "open settings".

General

  • Personality — Choose how iClaw communicates: Full (terse and sassy), Moderate, Neutral, or Custom (your own description, up to 280 characters)
  • Browser Bridge — Connect the browser extension for authenticated web access
  • Screen Context — Allow iClaw to capture screen content when asked
  • Action Consent — Control whether iClaw asks before performing actions. Destructive actions always require confirmation
  • Speech — Configure voice input and text-to-speech options

Models

Choose which AI backend iClaw uses for language understanding and response generation:

  • Apple Intelligence — Built-in, no setup required. Uses Apple’s on-device Foundation Models
  • Ollama — Uses locally-running open-source models via Ollama. Ollama must be installed and running. iClaw auto-detects available models and prefers ones already loaded in memory. Disabled when Ollama is not available

Both backends run entirely on-device. Your selection is remembered across launches.

Notifications

Three levels: Off, Basic, or Full. Controls heartbeat notifications for events, weather alerts, and memory maintenance reminders.

Automations

View, create, and manage scheduled recurring queries. Automations run on a schedule and deliver results via notification.

Skills

Enable or disable built-in and custom skills. Custom skills are loaded from ~/Documents/AgentSkills/.

Email

Configure Mail.app integration, including read and compose permissions.

Permissions

Dashboard showing the status of system permissions: Location, Contacts, Calendar, Reminders, Microphone, and more. Tap any permission to open the relevant System Settings pane.

History

Search and browse your conversation history. Messages are stored locally on your Mac.

About

App version, build information, and links.
