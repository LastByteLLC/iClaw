iClaw Help

Tool Chips & Shortcuts

While iClaw routes queries automatically, you can force a specific tool using tool chips — hashtag prefixes in your message.

Tool chips

Prefix your message with #toolname to bypass automatic routing and send it directly to that tool:

  • #weather — Weather
  • #stocks — Stocks
  • #timer — Timer
  • #calendar — Calendar
  • #calculator — Calculator
  • #convert — Currency/unit conversion
  • #translate — Translation
  • #news — News headlines
  • #search — Web search
  • #research — In-depth research
  • #podcast — Podcast search
  • #email — Email
  • #maps — Maps and directions
  • #help — Help

Example: #weather Paris next week

Ticker symbols

Type a dollar sign followed by a stock ticker to get a quick quote:

  • $AAPL — Apple stock price
  • $TSLA — Tesla stock price
  • $MSFT — Microsoft stock price

Tickers are validated before lookup — invalid symbols return an error rather than a bad result.

URL detection

Paste or type any http:// or https:// URL and iClaw automatically fetches and summarizes the page content.

Common company names

iClaw recognizes about 50 common company names and maps them to tickers automatically. Saying "How is Tesla doing?" routes to the Stocks tool for $TSLA.
