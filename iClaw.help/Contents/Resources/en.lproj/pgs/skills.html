Skills

Skills are lightweight extensions that add new capabilities to iClaw. They can fetch data from web APIs and present it conversationally.

Custom skills

You can add your own skills by placing Markdown files in:

~/Documents/AgentSkills/

Custom skills follow the AgentSkills.io format and are sandboxed to HTTP requests only. Enable or disable skills in Settings > Skills.

Using skills

Just ask naturally — iClaw matches your query against skill example phrases. Some skills also have hashtag handles (e.g. #crypto for Crypto Price).
